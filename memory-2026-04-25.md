# 2026-04-25 工作记录

## SRun Auto Login 项目

### 背景
用户是 IMUST 学生，需要 Magisk 模块自动登录校园网 SRUN 认证。

### 完成的工作
1. 分析了 Python 版 SRunPy-GUI-IMUST 的 SRUN 协议
2. 从零重写了 Go 版本 (v2.0.0 → v2.0.3)
3. 修复了多个关键 bug:
   - Android DNS 权限问题 (SELinux 拦截 Go DNS 解析器)
   - 校园域名内网解析 (shell 命令优先，DoH 兜底)
   - 客户端 IP 获取 (从网卡获取，不依赖 rad_user_info)
   - XXTEA 常量溢出、类型不匹配等编译问题
4. WebUI 全部重写 + 毛玻璃视觉设计
5. 源码推送到 GitHub: https://github.com/louisgoodabxd/jzzz

### 关键技术点
- Go 在 Android 上 DNS 解析被 SELinux 拦截 → 用 shell 命令 (getent/nslookup) 绕过
- 校园域名只能在校园网内解析 → shell 命令优先，公共 DNS 兜底
- 客户端 IP 从网卡获取 (UDP socket) → 与 Python 版 ip_utils.py 一致
- WebUI 用内置 HTTP 服务器 → 不依赖 KernelSU API
